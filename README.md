# la-grande-chorale-sainte-croix

Static React + Vite project for *La Grande Chorale de Sainte Croix*.

## Quick setup

1. Install:
```
npm install
```

2. Run dev server:
```
npm run dev
```

3. Build for production:
```
npm run build
```

4. Deploy to GitHub Pages (requires `gh-pages` and a repo):
```
npm run deploy
```

Upload the contents of this project to your empty GitHub repository (`https://github.com/michaelbruno1806/la-grande-chorale-sainte-croix`) and enable GitHub Pages from the `gh-pages` branch (or use the `deploy` script).
